CSCE 242: Website Applications

Description: CSCE 242 is a course designed to give students such as myself the knowledge and skills to develop a website. First, we will learn how to create the front-end with HTML and CSS. This includes aspects such as formatting text and images. Then we will use JavaScript to add another level to the website by allowing user interaction. This will teach the fundamentals to create a working website that serves an intended purpose. Below you will find all assignments related to these objectives.

Github Code Repository

Assignments

  1. Assignment 1: Basic HTML

    This assignment focuses on understanding some of the basic principles of HTML. Using these tags and elements it can create a basic website. This links to a website using tags such as headers, paragraphs, and tables from HTML to create a website on book recommendations.

  2. Assignment 2: Basic CSS

    This assignment is structured to learn how to style a website with CSS. It involves changes in formating, typography, and other aspects such as images which play a role in website design. This website shows some introductory CSS usage with a focus on strength training exercises.
