CSCE 242: Website Applications

Joseph Acree

CSCE 242 is a course designed to give students such as myself the knowledge and skills to develop a website. First, we will learn how to create the front-end with HTML and CSS. This includes aspects such as formatting text and images. Then we will use JavaScript to add another level to the website by allowing user interaction. This will teach the fundamentals to create a working website that serves an intended purpose. Below you will find all assignments related to these objectives.
